## Structure
 
```
ml-from-scratch/
├── 00-linear-algebra/
├── 01-probability/
├── 02-calculus/
├── 03-linear-regression/
└── ...
```
 
One folder per topic, numbered to preserve chronological order. Reading top-to-bottom retraces the learning arc.
 
## What's in each folder
 
Every topic folder contains three things:
 
- **`README.md`** — the writeup in plain English. What the concept is, why it matters, what clicked.
- **Implementation files (`.py`)** — the from-scratch implementation in NumPy.
- **`verify.py`** — sanity-check script comparing the from-scratch output against the equivalent library function.
If a folder is missing any of the three, the topic isn't done yet.
